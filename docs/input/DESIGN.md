## Overview

Web My Money design system — dark navy primary, Pearl Aqua teal accent.

## Style Foundations

- **Visual style:** premium, technical, confident, deep-space dark, fintech-grade
- **Typography fonts:** display=Satoshi (900 only), base=Inter (variable), label=Space Grotesk
- **Color palette:** background #080c14, surface #0f1624, primary #8fccb6, navy #1f2d56
- **Spacing scale:** confident density mode

## Colors

- **Primary (#8fccb6):** Pearl Aqua — CTAs, active states, success, highlights.
- **Secondary (#1f2d56):** Space Cadet Navy — section backgrounds, card surfaces.
- **Success (#8fccb6):** Pearl Aqua doubles as success.
- **Warning (#E89558):** Warm amber warning states.
- **Danger (#e05c6a):** Error states.
- **Surface (#0f1624):** Cards, panels, containers.
- **Text (#fbfbfb):** Near-white — never pure #ffffff.
- **Neutral (#080c14):** Page background — deep navy-black.
